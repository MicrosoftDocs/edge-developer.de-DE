# JsValueRef Typedef

Ein Verweis auf einen JavaScript-Wert.  
  
## Syntax  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## Hinweise  
 Ein JavaScript-Wert ist einer der folgenden Typen von Werten: undefined, NULL, boolescher Wert, Zeichenfolge, Zahl oder Objekt.  
  
## Anforderungen  
 **Kopfzeile:** jsrt. h
