# JsBoolToBoolean Funktion

Erstellt einen booleschen Wert aus einem `bool` Wert.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode JsBoolToBoolean(  
   _In_ bool value,  
   _Out_ JsValueRef *booleanValue  
);  
```  
  
#### Parameter  
 `value`  
 Der Wert, der konvertiert werden soll.  
  
 `booleanValue`  
 Der konvertierte Wert.
