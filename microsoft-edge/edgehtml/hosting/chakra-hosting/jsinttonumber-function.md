# JsIntToNumber Funktion

Erstellt einen Zahlenwert aus einem `int` Wert.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsIntToNumber(  
   _In_ int intValue,  
   _Out_ JsValueRef *value  
);  
```  
  
#### Parameter  
 `intValue`  
 Die `int` Konvertierung in einen Zahlenwert.  
  
 `value`  
 Der neue Zahlenwert.
