# JsGetTrueValue Funktion

Ruft den Wert `true` im aktuellen Skriptkontext ab.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsGetTrueValue(  
   _Out_ JsValueRef *trueValue  
);  
```  
  
#### Parameter  
 `trueValue`  
 Der `true` Wert.
